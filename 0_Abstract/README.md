# TITLE EXPLANATION

## WIPER CONTROL SYSTEM

The objective of this project is to design a wiper control system and to use it for security purpose

## SOFTWARE USED

* STM Cube IDE
* Qemu

## COMPONENTS USED

* ATmega328
* Switch
* LED
* Voltmeter
* Resister

## OUTPUT

* SWITCH OPEN -> LED ON
* SWITCH CLOSE -> LED OFF
